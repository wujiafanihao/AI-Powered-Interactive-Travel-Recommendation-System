# 交互式用户推荐查询脚本使用说明

## 📋 脚本功能

### `query_user_recommendations.py` - 交互式推荐查询

**功能**：通过账号密码登录后，输入你的需求（如"推荐北京适合亲子游的高分免费景点"），系统会根据你的画像和提问计算匹配评分并推荐景点。

**使用方法**：
```bash
cd /Users/mac/Desktop/Project/Python/xianyu/1/backend
/opt/anaconda3/envs/workspace/bin/python3 scripts/query_user_recommendations.py
```

**支持的需求类型**：
- 🏙️ 城市：北京、上海、广州、深圳、南京、杭州、西安、成都、重庆、武汉
- 🎯 类型：自然风光、历史文化、美食、购物、娱乐、博物馆、公园、动物园、游乐园
- 👨‍👩‍ 人群：亲子、情侣、朋友、老人、学生、一个人
- 🌤️ 季节：春、夏、秋、冬、四季
- 💰 门票：免费、不要钱
- ⭐ 评分：高分、评分高

## 🎯 匹配评分说明

### 综合得分计算
```
综合得分 = 50% × 搜索相关性 + 50% × 画像匹配度
```

- **搜索相关性**：根据景点评分（0-5 分）转换
- **画像匹配度**：根据用户画像（城市、旅行偏好、行为历史）计算

### 匹配度分数段
- 🟢 **高分景点 (60%+)**：非常符合你的画像
- 🟡 **中等景点 (40-60%)**：比较符合你的画像
- 🔴 **低分景点 (<40%)**：一般符合你的画像

## ⚠️ 注意事项

1. **条件不要太严格**：如果同时要求"北京 + 亲子 + 免费 + 高分 + 四季皆宜"，可能找不到景点
2. **使用默认账号**：测试账号 `test_admin` 的城市是北京，偏好是"自然风光，历史文化"
3. **查看真实用户**：建议使用数据库中 ID>140 的真实用户，他们的画像数据更完整
4. **退出程序**：输入 `quit`、`exit` 或 `q` 即可退出
